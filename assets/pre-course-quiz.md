# **React Crash Course 2025 – Pre-Course Quiz**

**Duration:** 20 minutes
**Format:** Multiple choice, true/false, short answer

---

## **Part 1 – Multiple Choice (1 point each)**

1. Which of the following is **true about `const` in JavaScript**?
   a) You can reassign the variable.
   b) You cannot reassign the variable, but objects and arrays can still be mutated.
   c) It behaves exactly like `var`.
   d) It cannot store objects or arrays.

2. What is the correct syntax for an **arrow function** that returns the square of a number?
   a) `function square(x) { return x*x; }`
   b) `(x) => x*x`
   c) `x => {x*x}`
   d) `const square = function x { x*x }`

3. Which HTML tag is used to link an external CSS file?
   a) `<script>`
   b) `<link>`
   c) `<style>`
   d) `<css>`

4. Which statement correctly describes **React**?
   a) A backend framework for Node.js
   b) A declarative JavaScript library for building UIs
   c) A CSS preprocessor
   d) A database management system

5. In React, **props are…**
   a) Mutable state variables
   b) Passed from parent to child, read-only
   c) Used to directly update parent state
   d) Only used for styling components
